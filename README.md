# Point Counter - Документация

## Описание проекта

**Point Counter** - это веб-приложение для подсчета объектов на изображениях путем интерактивной расстановки точек. Проект размещен на GitHub Pages и решает проблему визуального подсчета множества мелких объектов на складах и при инвентаризации.

**GitHub Pages**: [ссылка на проект]

## Решаемая проблема

Приложение предназначено для:
- Подсчета продукции на складах во время пересчетов и инвентаризации
- Работы с изображениями, содержащими множество мелких объектов
- Облегчения подсчета объектов, образующих ячейчатую структуру
- Упрощения зрительного восприятия и точного подсчета

## Технологии

- **HTML5, CSS3, JavaScript (ES6+)**
- **Bootstrap** - для интерфейса и иконок
- **LocalStorage** - для сохранения состояния
- **Canvas API** - для экспорта результатов
- **Модульная архитектура** - для поддержки кода

## Использование

### Базовые операции
1. **Загрузка изображения** - нажмите "Choose File" для выбора изображения
2. **Добавление точек** - кликайте по объектам на изображении
3. **Просмотр количества** - счетчик обновляется автоматически
4. **Настройка отображения** - используйте ползунки и чекбоксы
5. **Сохранение результата** - экспортируйте изображение с точками

### Управление историей
- **◀️ Undo** - отмена последнего действия
- **▶️ Redo** - повтор отмененного действия
- **🗑️ Reset** - полный сброс изображения и точек

## Установка и запуск
